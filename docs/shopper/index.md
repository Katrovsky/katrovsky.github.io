# Установка
---
Приложение распространяется следующими способами:

1. Для Android&trade;:

	<div class="grid cards" markdown>

	- [Google Play](https://play.google.com/store/apps/details?id=ru.sbmt.shopper)
	- [RuStore](https://www.rustore.ru/catalog/app/ru.sbmt.shopper)
	- [Huawei AppGallery](https://appgallery.huawei.com/#/app/C110698749)
	- Установка **`apk`** — можно скачать [здесь](Shopper_110.0.2.apk)
	
	</div>

	!!! IMPORTANT "Обратите внимание"
		Ссылки на apk обновляются автоматически, но с интервалом от 1 до 4 часов. При этом рекомендуемым способом установки и обновления является использование магазина приложения в вашем смартфоне.



2. Для iOS можно скачать [из AppStore здесь](https://apps.apple.com/ru/app/shopper-app/id6467191758).
